# 🎓 SEI2Hub (校园互助服务平台)

## 项目简介
SEI2Hub 是一个致力于解决大学校园内信息碎片化与信任机制缺失的综合性互助服务平台。学生可以在此方便地发布需求（快递代取、学习辅导、二手交换等）、发现匹配、建立信任并完成协作。
